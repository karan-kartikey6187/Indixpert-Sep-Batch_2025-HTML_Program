Full Stack Web Development

Frontend:
The part of a website or app that users see and interact with. (e.g., buttons, forms, design)
Backend:
The server-side of a website or app that handles logic, data, and requests. (e.g., login, calculations)
Database:
A storage system where all the data is kept for the website or app. (e.g., users, products, posts)

Frontend Development

HTML:
HTML is the language that creates the structure of a webpage. It defines headings, paragraphs, images, links, and other content.
CSS:
CSS is used to style and design a webpage. It controls colors, fonts, layout, spacing, and overall appearance.
JavaScript:
JavaScript makes a webpage interactive and dynamic. It handles actions like clicks, animations, form validation, and content updates.
